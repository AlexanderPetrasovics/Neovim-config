# Neovim-config
Eigene Neovim-config mit coc.

## Plug Installation
Standardpfad fuer Linux: 
curl -fLo ~/.local/share/nvim/site/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

## Coc-Plugins

- coc-tserver
- coc-emmet
- coc-marketplace
- coc-tailwindcss
- coc-css
- coc-html
- coc-json
- coc-phpls
- coc-vetur
- coc-vimtex
